# Breakout - JavaScript Practice
A reacreation of the popular game "Pong" in JavaScript, HTML, and CSS.

## Features
- Utilizes HTML Canvas and JavaScript to draw the game window and control the physics of the game.

## Usage
- Control the left paddle with the arrow keys. Bounce the ball back toward your opponent's side.

## Tech Stack
- **HTML5**
- **JavaScript (ES6+)**
